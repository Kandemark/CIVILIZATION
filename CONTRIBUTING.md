# Contributing to Civilization Simulation

Thank you for your interest in contributing!

## Getting Started
1. Fork the repository.
2. Clone your fork: `git clone https://github.com/yourusername/civilization.git`
3. Create a feature branch: `git checkout -b feature/amazing-feature`

## Coding Standards
- Use C99 standard.
- Follow the existing naming conventions (`civ_module_function`).
- Ensure all memory allocated is freed.
- Add comments for complex logic.

## Building
- Windows: Use `build_and_run.bat`.
- Linux/Mac: Use `make` (if available) or compile with `gcc`.

## Pull Requests
- Open a PR against the `main` branch.
- Describe your changes in detail.
- Ensure the game builds and runs without errors.
